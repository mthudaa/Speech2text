How to running the program
1. you must update your python version 3.1 or newer.
2. install all libraries for this program. open your terminal and type "pip install speechrecognition", "pip install pyttsx3", and then "pip install PyAudio"
3. if you lucky, "pip install PyAudio" can be working. However, if it command can't working. You can install "pip install pipwin" and then "pipwin install PyAudio"
